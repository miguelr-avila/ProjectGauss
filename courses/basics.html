Capitulo 1. La Naturaleza del Álgebra

Historia

DIOFANTO (325-409 D.C.) Famoso matemático griego quien fue perteneciente a la escuela de Alejandría. Fue el primer hombre en anunciar una teoría clara y concisa sobre las ecuaciones de primer grado. Al igual que ofreció fórmulas para la resolución de las ecuaciones de segundo grado.  Su obra mas conocida es Aritmética, debido a que ejerció influencia en el desarrollo del algebra entre los árabes como sobre la teoría moderna de los números.

 

¿Qué es el Algebra?

Álgebra: Es la rama de la matemática que estudia las cantidades, estructuras y relaciones de la forma más general posible. En otras palabras, el concepto de cantidad es mucho mas amplio en el algebra que en la aritmética, en esta última, las cantidades se representan con números y estos expresan valores constantes. En algebra, para llegar a la generalización de las cantidades, estas se representan por letras, las cuales pueden representar valores cambiantes, más conocidos como variables.

 

Expresión Algebraica: Es la forma en la que se conoce al conjunto de símbolos, números reales (constantes, en este caso números) y letras (variables, en este caso letras) para representar las cantidades, todo esto mediante los signos de operación.

 

  • Números: estos se emplean para expresar cantidades constantes
  • Letras: estas se emplean para expresar cantidades de todo tipo, tanto constantes como variables.

    •  

Signos en el algebra: Los signos utilizados en el algebra se dividen en tres clases:

 

  • Signos de Operación: En algebra se utilizan las mismas operaciones que en la Aritmética: Suma, Resta, Multiplicación, División, Elevación y Radicación. Los símbolos son:

 

  • Suma ( + )
  • Resta ( - )
  • Multiplicación ( x )
  • División ( / )
  • Potenciación ( ax )
  • Radicación ( √ ) 

    •  

  • Signos de Relación: Se emplean estos símbolos para indicar la relación que existe entre dos cantidades distintas

 

  • Igual a ( = )
  • Mayor que ( > )
  • Menor que ( < )

 

  • Signos de Agrupación: Estos signos indican que la operación colocada dentro de ellos se debe resolver primero.

 

  • Paréntesis ( )
  • Corchete [ ]
  • Llaves { }
